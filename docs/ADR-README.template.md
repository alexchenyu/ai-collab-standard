## 使用约定

- 新建 ADR 时先复制 `docs/ADR/000-template.md`，再按实际情况删减小节。
- 一条 ADR 一个文件，命名格式为 `NNN-slug.md`。
- `README.md` 只保留索引、状态和导航；不要再把完整摘要堆回这里。
- `lesson_learned.md` 只保留当前仍会反复使用的实现经验；ADR 历史和提案统一回到 `docs/ADR/`。
