# Lessons Learned

这里存放不适合继续堆进 `CLAUDE.md` 的内容：非显而易见的实现决策、跨模块契约、重要边界条件和长期维护经验。日常协作先看 `CLAUDE.md`，遇到细节问题再查这里；ADR 历史与提案统一放在 `docs/ADR/`。

## 使用约定

- 只记录“以后还会反复用到”的经验；一次性背景、显而易见的代码事实、API 枚举不要堆进来。
- 优先往已有主题下合并，避免按时间顺序无限追加。
- 如果某条已经进入 `CLAUDE.md` 或代码本身已经足够自解释，就从这里删掉。
